# WeRead
一款新闻阅读类app

一个Material Design风格的新闻阅读App，采用MVP模式，基于RxJava+Retrofit2进行开发。包含果壳热门，微信精选，知乎日报,和每天一个妹子图栏目。[点击下载APK][APK]
